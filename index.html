Project On Console

1: Open Visual Studio IDE software or download and install it from this link . 2: Create a new folder on your computer. 3: Open Visual Studio IDE and click on File > Open Folder, then browse and open the folder created in step 2. 4: Install Live-Server Extension if not already installed.

The console screen is used to find errors in your code, announce variables, and log important data down so you can see in your browser. You can annouce or insert anything in your console by using console.log(Then putting whatever you would like to announce within these brackets) In order to open the console screen on your browser, you must right click on the page you launched or any page. Then you must click inspect and head to the console tab. Then you will see the console.
